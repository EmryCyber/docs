Whether you're integrating with existing tools, setting up webhooks, or orchestrating complex workflows, Regentra provides the APIs and UI you need to move fast.

## Get started in three steps

<Steps>
  <Step title="Create your account">
    Sign up at [regentra.io](https://regentra.io) and log in to your dashboard. No credit card required to start.
  </Step>
  <Step title="Get your API key">
    Navigate to **Settings → API Keys** in your dashboard and generate your first API key. Keep it secure — you'll use it to authenticate all API requests.
  </Step>
  <Step title="Make your first API call">
    Use your API key to authenticate and interact with Regentra. See the [Quick Start](/quickstart) guide for a working example.
  </Step>
</Steps>
